intellij idea 快捷键

  • alt+1 ->快速定位到project structure 、然后按esc回到编辑处。
  • esc ->在其他窗口中，按esc键都能够快速切换到编辑出
  • alt+insert-> 新建文件 alt+insert 选择class 、file…..
  • alt + insert ->万能生成键 ，自己在编辑器中按下看看其选项
  • psvm -> System.out.println(“”);
  • ctrl+w 将选中对应的单词 连按w键会延伸。
  • ctrl+shift+enter-> new ArrayList() ，注意按下快捷键会自动补全分号的。其实语法结构的括号其也能补全。
  • alt+enter ->类似eclipse中的本地变量自动生成一样，比如会自动补全ArrayList arrayList的声明。
  • alt+/-> 自动补全当前上下文中出现过的单词
  • .for+tab-> arrayList.for 自动生成响应的语法格式代码
  • ctrl+j ->针对上面的快捷键 可以通过 ctrl+j 查看intellij idea中对应的模板代码设置
  • ctrl+o-> 重写父类方法
  • ctrl+i-> 实现接口方法
  • ctrl+p-> 方法参数提示
  • ctrl+space -> 方法提示
  • ctrl+alt+s-> intellij idea settings
  • ctlr+alt+shift+s或F4-> project settings
  • ctrl+shfit+F10 ->启动main方法
  • alt+数字0、1、2、3、4…..->各种透视图模块的定位
  • ctrl+tab -> 各种视图间的切换

上面是我个人总结的常用快捷键，其实这类文章有很多，一坨坨的，我的独一份儿，供大家参考。

secure Crt 颜色配置

SecureCRT：SecureCRT配色方案

  1. SecureCRT：SecureCRT配色方案

  2. securt默认安装好后只会显示粗体，不能带颜色。
    我们可以自定义带颜色的显示，调整字体为等宽字体。

securecrt的控制台颜色设置：
Options ->Session Options -> Terminal -> Emulation 中Terminal选择ANSI or linux or xterm,必须钩上 ANSI Colour

securecrt字体和大小调整：
Options->SessionOptions->Appearance->font
软后选择Consolas 大小调整为16pt比较舒服。

调整完整后securtcrt虽然显示颜色了，但是蓝色字体看不清。
securecrt蓝色看不清解决办法：
Options–>Session Options–>Terminal –>Appearance–>ANSI color–>点掉使用默认颜色->bold color(B)–>将看不清的蓝色调浅

总之第一步中未达到效果的，请将bold的也设置一遍就好了
