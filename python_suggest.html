why python ?

Life is short ,we use python

上句的流行就如 php是世界上最好的语言一样，但是 相对于php的嘲讽，Life is short ,we use python 真的是一种赞美。我的个人经验来看，python是老少皆宜的一门计算机语言，入门简单，功能库丰富强大，适合做快速开发，适用于各个领域，运维 脚本，web开发，机器学习等等，当然在性能方面确实差了点，但这快速出活儿的时代，先能做出来是很重要的一件事儿。这样的权衡，自己拿捏。

拓展： python 哲学 –> 在python终端中 import this 下，你会看到什么是优雅。
