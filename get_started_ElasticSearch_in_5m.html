ElasticSearch可以很容易地搭建运行一个全功能的搜索服务器。事实上，它很容易，给我5分钟时间，让你体验到它的强大。

实战

1. 安装运行ElasticSearch

本教程，我会认为你是在Linux或Mac环境。 你也应该有JDK 6或以上安装。

wget https://download.elastic.co/elasticsearch/elasticsearch/elasticsearch-1.7.2.tar.gz
tar -zxvf elasticsearch-1.7.2.tar.gz
cd elasticsearch-1.7.2
bin/elasticsearch

你应该可以在terminal中看到如下相关内容

再看几个例子

  1. 不包含search条目的文章

curl 'http://node01:9200/blog/post/_search?q=-title:search&pretty=true'

2.检索包含search 但不包含distributed的文章
curl 'http://node01:9200/blog/post/_search?q=+title:search%20-title:distributed&pretty=true&fields=title'

  1. 范围搜索
curl -XGET 'http://localhost:9200/blog/_search?pretty=true' -d '
{ 
    "query" : { 
        "range" : { 
            "postDate" : { "from" : "2011-12-10", "to" : "2011-12-12" } 
        } 
    } 
}'

es network setting
