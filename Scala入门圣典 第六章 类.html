类的定义

最简单的类定义

//简单的类写法
class Animal {

}

//伴生对象，与类名必须一样，方法为静态，单例
object Animal {
  def main(args: Array[String]) {
    println("hello intellij idea")
  }
}

带有字段、方法、函数的简单类

class Counter {
  private var value = 0 //字段必须初始化

  def increment() {
    value += 1
  }

  def current() = value
}

object Counter {
  def main(args: Array[String]) {
    val counter: Counter = new Counter
    for (i <- 1 to 10) {
      counter.increment()
    }
    println(counter.current())
  }
